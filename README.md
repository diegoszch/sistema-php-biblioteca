sistema-php-biblioteca
======================

Sistema simples desenvolvido em PHP, que visa auxiliar os novos praticantes da linguagem.

Configuração
============

Arquivo de configuração do banco 

global.php
$db_host = "localhost";
$db_user = "root";
$db_pass = "1234";
$db_name  = "biblioteca";

Local do banco de dados
sql\biblioteca.sql
